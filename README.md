# 까치 (kkachi)

AI 일정 비서 '까치'를 만드는 저장소입니다.

## 구조

| 폴더 | 내용 |
|---|---|
| `context/` | 까치 컨텍스트 원본 — 클로드 스킬(SKILL.md) 형태로 축적 |
| `app/` | 까치 앱 코드 (Expo) |

## 컨텍스트

까치에 관한 맥락은 `context/` 아래 스킬로 축적합니다.
각 스킬의 프론트매터(name·description)만 보고 필요한 맥락을 골라 로드할 수 있습니다.

| 스킬 | 내용 |
|---|---|
| `context/kkachi-brand` | 브랜드 — 이름·미션·가치, 컬러, 타이포, 보이스 |
| `context/kkachi-design` | 디자인 시스템 — 토큰(컬러·타이포·radius·shadow), 컴포넌트 스펙, 다크모드 TODO |
| `context/kkachi-product` | 제품 정의 — 타깃, 결정적 순간, 앱 구조(5탭), 개념 경계, UX 원칙 |
| `context/kkachi-business` | 사업·출시 — 수익 모델, 출시 전략·시점, 준비물 |
| `context/kkachi-tech` | 기술 — 확정 스택, 데이터 방침, 아키텍처 |

이 저장소가 까치 컨텍스트의 원본입니다. 코워크 등에 설치된 사본과 내용이 갈리면 이 저장소를 기준으로 맞춥니다.

## 개발

2026-07-24부터 개발 단계입니다. 앱 개발은 `app/`에서 하며, 제품·기술 결정은 `context/`의 스킬을 따릅니다.
컨텍스트와 구현이 갈리면 컨텍스트를 먼저 고친 뒤 구현을 맞춥니다.

개발 세션은 시작 전에 `context/`의 다섯 스킬을 모두 읽습니다. 개발 순서(첫 마일스톤)와 환경 준비물 체크리스트는 `context/kkachi-tech`에 있습니다.
